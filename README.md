# Word to PDF Converter

## 📌项目简介

这是一个 Python 脚本，用于将 `.docx` 文件转换为 `.pdf` 文件。支持单个文件转换或批量转换整个目录下的所有 `.docx` 文件。

## 功能

* 图形界面（GUI）操作
* 命令行（CLI）操作
* 将单个`.docx`文件转换为`.pdf`文件
* 批量转换指定目录下的所有 `.docx` 文件
* 错误处理：自动跳过非 `.docx` 文件，并显示转换失败的原因

## 👀️ 安装依赖

在使用脚本之前请确认已安装Python3和以下库：

```
pip install docx2pdf argparse tkinter
```

**注意**：`tkinter`通常是Python内置库，如果缺失请参考系统文档安装

## 🚀️ 使用方法

### 1.图形界面模式（GUI）

在克隆的项目文件夹下直接运行程序可启动图形界面：

```
python .\word2pdf.py
```

界面功能：

* 通过“浏览”按钮选择单个文件或文件夹
* 点击“转换”按钮执行单个文件转换
* 点击“批量转换”按钮处理整个文件夹
* 底部状态栏显示实时进度

### 2.命令行模式（CLI）

保持原有命令行操作方式：

#### 转换单个文件：

```
python .\word2pdf.py [文件路径.docx]
```

示例：

```
python .\word2pdf.py "C:/Documents/example.docx"
```

#### 批量转换目录下的所有 `.docx` 文件:

```
python .\word2pdf.py [目录路径]
```

示例：

```
python .\word2pdf.py "C:\Documents\pWordFiles"
```

## 📝 注意事项

1. 文件格式：
   * 仅支持`.docx`格式，不支持旧版`.doc`
   * 非.docx文件会被自动跳过
2. 输出位置：
   * 转换后的PDF会保存在与原文件相同的目录下
   * 文件名保持与原文件一致
3. 路径规范：
   * 包含空格或特殊字符时请使用引号包裹路径
   * 示例：`"path with spaces/文件.docx"`
4. 权限要求：
   * 确保对目标目录有写入权限
   * 转换时请关闭正在编辑的Word文件

## ❓ 常见问题

### Q: 转换失败怎么办？

* 确保输入的文件是有效的 `.docx` 文件。
* 检查文件是否被其他程序（如 Word）占用。
* 查看错误信息，确认是否有权限问题。

### Q：如何查看详细错误信息？

* GUI模式：错误信息会显示在弹窗中
* CUI模式：错误信息会输出到控制台

### Q：能否转换子目录中的文件？

* 当前版本仅处理指定目录下的文件，不会递归处理子目录
